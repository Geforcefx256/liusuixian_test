# 当前分支 vs origin/agent-V2-base 的 SSO 配置差异分析计划

日期：2026-03-27

## 目标
对比当前分支与远端 `origin/agent-V2-base` 在 SSO/认证配置上的差异，重点检查：
- `config.json` 中与 SSO、认证模式、提供商、回调地址、会话治理相关的字段
- 认证控制器、路由、服务层实现差异
- 当前分支缺失了哪些原分支中依赖配置驱动的功能

## 执行步骤
1. 搜索当前分支和远端分支中的 `sso`、`auth`、`oidc`、`oauth`、`provider`、`callback`、`config.json` 相关实现。
2. 分别读取当前与远端的关键配置文件和认证入口代码。
3. 汇总 `config.json` 字段差异，以及这些字段在代码中的使用位置。
4. 输出：
   - 当前缺失的配置项
   - 当前退化或缺省的功能行为
   - 如需恢复原能力，最小应补哪些配置和模块
